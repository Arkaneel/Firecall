## Running This Project

To run this project,make sure NodeJs is Installed Then Follow The Steps..

* First Edit src/server/firebase.js and add Your Config


* Locate The Directory...
```bash
  cd <directory>
```
```bash
  npm install
```
```bash
  npm start
```
